    Enable local node on this phone
    1. Choose Local node mode above (pruned, full, or mesh).
    2. Tap Start full node — runs bloodstoned without mining on this phone.
    3. Wait for chain sync — progress bar shows download status.
    4. Optional: create an on-device wallet and tap Use for mining.
    5. On Bitaxe/cpuminer: paste the LAN host + port from the section above (same Wi‑Fi only).
    6. Optional: enter a STONE payout address and tap Start offline mining to mine on this device too.
